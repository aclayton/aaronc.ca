What is it

Svg Machine is a single page app (SPA) that I made after reading a great article by Chris Coyier of Css Tricks on building svg sprite files using the <defs>, and <use> tags.

I really like the idea of having all of a site's icons in an easy to serve definition file which can also act as a self contained library, with the ability to add and extract individual icons. However, to make this approach more user friendly requires an easy to use library manager.

To complete the project the ability to live edit the individual svgs was added so all that post export svg clean up (you know I mean) is a snap.
